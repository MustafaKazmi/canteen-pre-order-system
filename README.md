# Smart Canteen Pre Order System

## Status

Currently in active development. Core features like UI design and database integration are in progress.

A web-based application that allows students to order food from the canteen in advance, helping reduce waiting time and improve overall efficiency.

## Features

* User registration and login
* Food ordering system
* Database integration for storing user and order data
* More features can be added as the project grows

## Tech Stack

* JavaScript
* HTML, CSS
* MySQL

## Objective

To reduce crowding and waiting time in college canteens by allowing students to place their orders ahead of time.

## Development Roadmap
  Design UI (HTML, CSS)
  Implement login system
  Connect database
  Build ordering system
